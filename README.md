Spring MVC based Online Admission System
-------------------

Technologies used-

1. Spring - Core, AOP, MVC, ORM, Security
2. HTML5, CSS3, Bootstrap. jQuery
3. Hibernate
4. Postgres/H2
5. Redis for Session Management

Redis Server can be downloaded from here
https://chocolatey.org/api/v2/package/redis-64/3.0.501
localhost and default port is used here
